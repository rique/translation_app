# Translation Frontend

A React-based interface for managing multi-language translations.

## Features
- File sidebar for quick switching between templates.
- Side-by-side translation grid (English/Spanish/French).
- Automatic syncing with the Python backend.

## Tech Stack
- **React**: UI Framework.
- **Tailwind CSS**: Styling (via CDN in `public/index.html`).
- **Axios**: API communication.
- **Lucide-React**: Icon set.

## Running
Run `npm start` to launch on `http://localhost:3000`.
